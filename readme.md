# Angular2 Postit Notes

## Installation
`npm install`

## Running
`npm start`

## Digging in

- `src/components`
    + `postit-notes`: the main postit-notes container
    + `postit-note`: a component for displaying a single postit
- `src/models`
    + `postit`: the model definition and API binding for saving/loading postits
